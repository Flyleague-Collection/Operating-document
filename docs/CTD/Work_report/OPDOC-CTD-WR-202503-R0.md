本季度下旬空管中心开设了“管制员培训中心·学习中心 CTD·Learning Center”网站，已完成8/33个知识点。同时，我们将教员、教师和学员三者之间的关系进行调整，强调省去负责环节，保留简洁部分。我们学习并创建了小班教学体系，即：每一位教员直接负责一期学员，并设置固定周期的课程，保证学习进度统一。
